COVID-19 Dashboard
Portafolio

Métricas Incorporadas

Criterio para medir los casos activos a nivel nacional y regional.
Criterio para comparar los casos activos vs los casos recuperados.
Criterio para comparar los casos confirmados a nivel regional y comunal.
Criterio para medir el indicador de carga.
Criterio para medir la toma de PCR por cada 1000 habitantes.
Criterio para medir el R efectivo.
Criterio para medir el tiempo de examen y laboratorio.
Criterio para medir la trazabilidad de la enfermedad.

Descripción

Proyecto creado en la asignatura "Taller de Integración 2", consiste en elaborar una dashboard cuyo objetivo es mostrar o visualizar de mejor manera la evolución de la enfermedad COVID-19 en Chile. El proyecto base es propiedad de Alonso Silva, solo que se le incorporaron nuevas métricas (las que se visualizan en el carusel de imágenes).

Tecnologías involucradas: Python y Streamlit
